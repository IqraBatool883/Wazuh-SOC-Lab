
# 📌 Project Overview

This repository contains my **Week 1 SOC Internship Project**, where I built a beginner-friendly **Security Operations Center (SOC) Lab** using **Wazuh SIEM** in a virtual environment.

The project demonstrates the complete setup of Wazuh, deployment of agents, security monitoring, alert generation, and File Integrity Monitoring (FIM).

---

# 🎯 Objectives

- Install Ubuntu Server
- Install Kali Linux
- Install Windows Virtual Machine
- Deploy Wazuh SIEM
- Configure Wazuh Manager
- Deploy Wazuh Agents
- Generate Security Alerts
- Configure File Integrity Monitoring (FIM)
- Verify Alerts through the Wazuh Dashboard

# 📋 Project Tasks

## 1️⃣ Virtual Machine Setup

Created three virtual machines:

- Ubuntu Server
- Kali Linux
- Windows

Configured networking and verified communication between all machines.

---

## 2️⃣ Wazuh Installation

Installed:

- Wazuh Manager
- Wazuh Indexer
- Wazuh Dashboard

Verified that all services were running successfully.

---

## 3️⃣ Agent Deployment

Installed Wazuh Agent on:

- Kali Linux
- Windows

Verified that both agents successfully connected to the Wazuh Manager.

---

## 4️⃣ Security Alert Generation

Generated an SSH Authentication Failure from Kali Linux.

Verified:

- Log Collection
- Event Detection
- Security Alert Generation
- Alert Visibility in Wazuh Dashboard

---

## 5️⃣ File Integrity Monitoring (FIM)

Configured File Integrity Monitoring on:

### Linux

- `/etc`

### Windows

- `System32`

Performed:

- File Creation
- File Modification
- File Deletion

Verified that Wazuh generated alerts for every change.
